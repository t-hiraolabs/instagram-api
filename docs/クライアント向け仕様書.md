## 1. プロジェクト概要

### 1.1 背景・目的

Instagram運用において、日々のコンテンツ制作・投稿管理に多大な時間と労力がかかっているという課題を解決するため、AI技術を活用したInstagram投稿管理アプリ「InstaAI」を開発します。

個人事業主・中小企業のInstagram担当者が、専門知識がなくても高品質なコンテンツを効率的に作成・配信できる環境を提供することを目的とします。

### 1.2 対象ユーザー

- 美容サロン、カフェ、アパレルなどの個人事業主
- SNS専任担当者のいない中小企業
- Instagram運用を効率化したいビジネスオーナー

### 1.3 提供形態

- **スマートフォンアプリ**（iOS / Android）
- **Webブラウザ**（PC・スマートフォン）

## 2. 機能一覧

### 2.1 主要機能

#### ① AIコンテンツ生成機能

| 機能 | 詳細 |
|------|------|
| テキストからキャプション生成 | 業種・トーン・季節テーマを指定してAIがキャプションとハッシュタグを自動生成 |
| 写真からキャプション生成 | 投稿したい写真をアップロードするとAIが内容を分析してキャプションを生成 |
| ストーリー用コンテンツ生成 | ストーリー専用の短文・アンケート・カウントダウン等のコンテンツを生成 |
| リール用キャプション生成 | 動画の各スライドに合わせたキャプションを複数生成 |
| キャプション改善提案 | 既存のキャプションをAIが3パターンに改善提案 |
| ハッシュタグ自動生成 | 投稿内容に最適なハッシュタグを15〜20個自動生成 |

**AI生成カスタマイズ項目**

- **業種**: 美容、カフェ、アパレル、飲食店、ホテル、フィットネス など13種類
- **投稿トーン**: 明るい / プロフェッショナル / カジュアル / 感情的 / ユーモラス
- **季節テーマ**: お正月、バレンタイン、花見、夏祭り、クリスマス など年間イベント対応

#### ② 投稿管理機能

| 機能 | 詳細 |
|------|------|
| 投稿スケジュール設定 | 日時を指定して投稿を予約、自動で配信 |
| 下書き保存 | 作成途中のコンテンツを保存・後から編集 |
| 投稿一覧管理 | 予約中・下書き・公開済み・失敗を一覧で確認 |
| 即時投稿 | 作成したコンテンツをすぐにInstagramへ投稿 |
| 繰り返し投稿 | 毎日・毎週・毎月・平日のみ など定期投稿設定（Proプラン） |

**対応投稿タイプ**

- フィード投稿（単体・複数枚カルーセル）
- ストーリー（画像・動画）
- リール（動画）

#### ③ ストーリー・リール作成機能

| 機能 | 詳細 |
|------|------|
| ストーリーエディター | テキスト追加・背景色変更・レイアウト調整 |
| リールスライド作成 | 複数スライドに見出しテキストを追加 |
| テンプレート選択 | あらかじめ用意されたデザインテンプレートから選択 |

#### ④ メンバー管理機能（ロスター機能）

| 機能 | 詳細 |
|------|------|
| スタッフ登録 | 名前・写真を登録してメンバー一覧を管理 |
| 本日の出勤表示 | 出勤メンバーをビジュアルで表示・Instagram用画像として出力 |

#### ⑤ Instagram連携機能

| 機能 | 詳細 |
|------|------|
| Instagram アカウント連携 | ビジネスアカウント・クリエイターアカウントをOAuth認証で安全に接続 |
| 投稿分析データ取得 | インプレッション・リーチ・エンゲージメントをアプリ内で確認 |

#### ⑥ アカウント・ブランド設定

| 機能 | 詳細 |
|------|------|
| ブランド設定 | ブランド名・業種・ターゲット層・投稿トーンを登録 |
| プロフィール管理 | メールアドレス・パスワード変更 |
| 利用状況確認 | AI生成の月次使用回数・残り回数を確認 |

## 5. システム構成

### 5.1 利用技術・サービス

| 区分 | 内容 |
|------|------|
| アプリ開発 | React Native（iOS/Android/Web対応） |
| AI | Anthropic Claude API |
| SNS連携 | Instagram Graph API（Meta公式） |
| データベース | Supabase（PostgreSQL） |
| ファイル保存 | Supabase Storage |
| 認証 | Supabase Auth |
| Webホスティング | Vercel |

### 5.2 セキュリティ対策

- ユーザーデータは行単位のアクセス制御（RLS）で保護
- InstagramのOAuth認証による安全なアカウント連携
- APIキー・トークンはサーバーサイドで管理（クライアントに露出しない）
- AI利用制限をデータベースレベルで二重管理

## 8. 制約事項・前提条件

1. Instagramへの投稿には **ビジネスアカウントまたはクリエイターアカウント** が必要です（個人アカウント不可）
2. AI生成にはインターネット接続が必要です
3. 動画投稿はInstagram側の処理時間（数秒〜数分）が発生します
4. ストーリーはInstagramの仕様上、**24時間後に自動削除**されます
5. Instagram Graph APIの仕様変更により、一部機能が影響を受ける場合があります
